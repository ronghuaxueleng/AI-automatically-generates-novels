# Lorebook和记忆管理功能提示词

## Lorebook功能提示词

```
Lorebook - Novel.ai风格的世界观管理系统

功能描述：
- 自动从生成的内容中提取和管理世界观元素
- 分类支持：世界观、角色、地点、事件、物品、概念等
- 可以为每个条目设置关键词、优先级和上下文窗口
- 内容自动关联到相关章节
- 支持导入导出功能

使用方法：
1. 通过"保存到Lorebook"按钮将章节内容添加到Lorebook
2. 在Lorebook面板中管理各种世界观元素
3. 手动添加或编辑条目
4. 设置关键词以便内容匹配
5. 调整优先级控制内容的重要性
6. 启用或禁用特定条目

技术实现：
- 使用localStorage存储Lorebook数据
- 实现自动提取和分类算法
- 支持按类别和关键词过滤
- 提供友好的用户界面进行管理
```

## 记忆管理器提示词

```
Memory Manager - 增强的上下文记忆系统

功能描述：
- 自动记忆和管理小说的重要信息
- 记忆类型：章节记忆、角色记忆、事件记忆
- 支持记忆压缩和自动摘要
- 可配置记忆优先级和长度
- 增强生成内容的一致性

设置选项：
- 最大记忆章节数
- 记忆压缩比例
- 优先记忆最新内容
- 自动摘要阈值

工作原理：
- 自动提取和整合关键信息
- 对长内容进行压缩和摘要
- 为AI生成提供上下文记忆
- 维护故事的连贯性和一致性

技术实现：
- 使用localStorage存储记忆数据
- 支持服务器同步
- 内置文本分析和摘要算法
- 提供记忆数据的导入导出
```
